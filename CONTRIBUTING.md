## 3\. Langkah 3: Bikin Branch Fitur di Fork Kamu

Buka folder `Rupi-Wallet` yang udah kamu fork di terminal, terus bikin branch buat isumu:

```

```

> Ganti `{NOMOR_ISUMU}` dengan nomor ID isumu, ya.
